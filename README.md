**TOLL is an ERC20 Token that respresents all the Fees Paid in the Ethereum Network.** Toll allows you to recover your network fees.

 * Reclaim Previously Spent Fees (Only during Presale).
 * Reclaim Token Transfer Fees (ERC20 Tokens Only).
 * Reclaim Uniswap Exchange Fees.
 * Reclaim Sushi Swap Exchange Fees
 * Recliam other Defi Fees (comming soon).
